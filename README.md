# Cloud Sail Portal (Frontend MVP)

## Start

```bash
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env.local` and adjust values:

- `NEXT_PUBLIC_SITE_URL`
- `API_BASE_URL`
- `NEXT_PUBLIC_API_BASE_URL`

## Scope

This project contains only the corporate website frontend:

- Next.js App Router
- TypeScript
- Tailwind CSS
- SEO pages (`sitemap`, `robots`, metadata)
- Lead submit route handler proxy (`/api/leads`)
